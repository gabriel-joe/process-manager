# process-manager
An Example of one application using Spring Boot + Vue.js

## Pre requirements
- Install nodeJS

## To start the back-end you need to do these steps:
- Enter folder process-manager-app 
- run this command "./gradlew bootRun"

## To start the back-end you need to do these steps:
- Enter folder process-manager-app 
- run this command "npm run dev"
